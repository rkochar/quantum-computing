Simon’s algorithm

In this exercise you will implement the complete Simon’s algorithm including the postprocessing. More concretely you will complete a function that

  • takes as input UfUf
  • implements the quantum circuit of Simon’s algorithm
  • runs the quantum circuit with the QASM backend a large enough number of times
  • returns both the quantum circuit and the secret string ss
  • the secret string ss should be returned as a list of integers with the convention [sn−1,…,s0][sn−1,…,s0]

UfUf is a unitary acting on a 2n2n qubit register. More precisely, given

f:{0,1}n→{0,1}nf:{0,1}n→{0,1}n

and two integers
x=∑i=0n−12ixi, y=∑i=0n−12iyix=∑i=0n−12ixi, y=∑i=0n−12iyi

the unitary UfUf performs the transformation
|yn−1⟩…|y0⟩|xn−1⟩…|x0⟩↦|yn−1⊕f(x)n−1⟩|y0⊕f(x)0⟩|xn−1⟩…|x0⟩ ,|yn−1⟩…|y0⟩|xn−1⟩…|x0⟩↦|yn−1⊕f(x)n−1⟩|y0⊕f(x)0⟩|xn−1⟩…|x0⟩ ,
where qubit 0 in the register corresponds with x0x0 and f(x)if(x)i is the ii-th bit of f(x)f(x).

Rubric

  • 60% points correct implementation of Simon’s quantum circuit
  • 40% points correct classical postprocessing that returns the string ss
